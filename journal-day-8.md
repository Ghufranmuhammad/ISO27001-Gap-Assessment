# Day 8 Journal — ISO 27001 Gap Assessment

Today I completed my first ISO 27001 gap assessment simulation.

### Key Skills Learned:
* Identify compliance gaps between current and desired state
* Map ISO 27001 controls to organizational processes
* Define scope for an assessment
* Understand how auditors evaluate ISMS readiness

The most challenging part was understanding how to translate abstract ISO clauses into real-world organizational controls. This exercise helped me think like a GRC analyst working in a real audit environment.
