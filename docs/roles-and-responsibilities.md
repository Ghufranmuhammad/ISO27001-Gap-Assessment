Roles and Responsibilities

CEO

- Approves the Information Security Policy.
- Provides resources for the ISMS.
- Supports continual improvement.

---

IT Manager

- Implements technical security controls.
- Manages system security.
- Supports incident response.

---

GRC Analyst

- Conducts risk assessments.
- Maintains ISMS documentation.
- Supports audits and compliance activities.

---

Employees

- Follow security policies.
- Protect company information.
- Report security incidents.

---

HR Manager

- Coordinates security awareness training.
- Ensures access is removed when employees leave.
- Supports background screening where applicable.
