# Human Risk Scenarios & Remediation Reference

| Risk Scenario | Impact Summary | Recommended Controls | Verified Audit Evidence |
| :--- | :--- | :--- | :--- |
| **1. Phishing Exploitation** | Compromised credentials leading to a wide-scale cloud environment breach. | Inbound email filtering sandboxes, phishing simulations, hardware MFA. | Phishing simulation click dashboards and SIEM MFA logging. |
| **2. Weak Passwords** | Local service account spray attacks enabling lateral hacker expansion. | Single Sign-On (SSO), automated validation rules, password length requirements. | IAM active directory group policy settings printouts. |
| **3. Tailgating / Badging** | Unauthorized entry to restricted server infrastructure. | Mantrap facilities, physical badges, guard stations, reporting training. | Visitor sign-in registers and server facility CCTV recording clips. |
| **4. Social Engineering** | Unauthorized balance transfers via spoofed vendor calls. | Mandatory out-of-band phone/video validation guidelines for transfers. | Executed multi-signer accounting workflow transaction files. |
| **5. Lost/Stolen Laptop** | Potential unencrypted local hard disk parsing by third parties. | Centralized Mobile Device Management (MDM), BitLocker enforcement. | MDM inventory list showing 100% full-disk encryption compliance. |
| **6. Misdirected Emails** | Sending high-confidential client financials to a public vendor. | Data Loss Prevention (DLP) filters scanning outbound attachment attributes. | Automated DLP quarantine alert summaries and compliance logs. |
| **7. Clean Desk Violation** | Password cheat sheets left visible on unattended monitors. | Periodic after-hours office security audits with minor citations. | Completed facility security patrol checklist logs. |
| **8. Malicious Insider** | Departing salesperson downloading proprietary client lists. | USB mass storage bans, strict role-based access limits, behavior analytics. | DLP storage device exclusion registry configurations. |
| **9. Shoddy Work-from-Home** | Smart home IoT devices snooping on sensitive audio calls. | Always-on corporate VPN routing configurations, privacy headsets. | Firewall tunnel tracking session histories. |
| **10. Unreported Incidents** | Malware spreading on endpoint systems for weeks without notice. | One-click reporting hotkeys, active non-punitive reporting cultures. | SOC tracking queues verifying recorded incident metrics. |
